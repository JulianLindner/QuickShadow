# What is it?
This plugin adds a processing algorithm that lets you create cast shadows from polygon shapes.

# How to Use
Use the UI Button or the algorithm "Create cast shadow" to open the algorithm window.
<img width="926" height="855" alt="grafik" src="https://github.com/user-attachments/assets/9aeeee33-40ff-47ab-80d5-3a59fa22d710" />
Select the input layer.

Define the height field. This could be either the stories of buildings or height in meter.
If the height field is kept empty, all shadows just get the same length.

Define the angle of the shadow.

Define the length factor that is multiplied with the height field to create the length of the shadow.
